# 访客子系统数据模型

## visitor_appointment

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint | 预约单 ID |
| visitor_name | varchar | 访客姓名 |
| visitor_phone | varchar | 访客手机号 |
| host_user_id | bigint | 被访人 ID |
| visit_start_at | datetime | 预约开始时间 |
| visit_end_at | datetime | 预约结束时间 |
| status | varchar | draft / pending / approved / rejected / cancelled / expired / checked_in / checked_out |
| qr_code_id | bigint | 二维码凭证 ID |

## visitor_access_record

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint | 通行记录 ID |
| appointment_id | bigint | 预约单 ID |
| direction | varchar | in / out |
| gate_id | bigint | 门禁点位 ID |
| occurred_at | datetime | 通行时间 |
